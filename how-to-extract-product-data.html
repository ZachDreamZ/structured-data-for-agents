How to Extract Product Data from Any URL

A practical guide for developers building AI agents, price trackers, or data pipelines that need structured product information from web pages.

Why Extract Product Data?

If you're building an AI agent, price comparison tool, or e-commerce aggregator, you need structured product data — name, price, description, images, availability — from arbitrary web pages. The challenge: every site has different HTML, anti-bot measures, and selector maintenance overhead.

Writing custom scrapers for each site is expensive and fragile. A single site redesign breaks everything. Here's a better approach.

The API Approach

Instead of scraping HTML and parsing selectors, use a structured extraction API. Send a URL, get validated JSON back.

1 Send a POST request with the target URL and schema type.
POST https://agentapigw.dpdns.org/v1/extract Content-Type: application/json { "url": "https://example.com/product/some-widget", "schema_type": "product" }
2 Get structured JSON with validated fields.
{ "name": "Some Widget Pro", "price": 49.99, "currency": "USD", "description": "A high-quality widget for...", "images": ["https://example.com/img/widget.jpg"], "availability": "in_stock", "brand": "ExampleCo", "sku": "WIDGET-PRO-001", "rating": 4.5, "review_count": 128 }
3 Handle errors gracefully. If extraction fails, the API returns clear error messages.
{ "error": "extraction_failed", "message": "Page returned 403 — anti-bot protection detected" }

Supported Schema Types

The API supports three schema types for different use cases:

Product — Name, price, currency, description, images, availability, brand, SKU, ratings. Use this for e-commerce pages.

Article — Title, author, published date, content, word count, language, tags. Use this for news sites and blogs.

Company — Name, description, industry, employees, founded, headquarters, social links. Use this for business directories.

Integration Examples

Python

import requests resp = requests.post( "https://agentapigw.dpdns.org/v1/extract", json={"url": "https://example.com/product", "schema_type": "product"}, headers={"Authorization": "Bearer YOUR_API_KEY"} ) data = resp.json() print(f"Product: {data['name']} — ${data['price']}")

JavaScript

const res = await fetch("https://agentapigw.dpdns.org/v1/extract", { method: "POST", headers: { "Content-Type": "application/json", "Authorization": "Bearer YOUR_API_KEY" }, body: JSON.stringify({ url: "https://example.com/product", schema_type: "product" }) }); const data = await res.json(); console.log(`Product: ${data.name} — $${data.price}`);

cURL

curl -X POST https://agentapigw.dpdns.org/v1/extract \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"url":"https://example.com/product","schema_type":"product"}'
