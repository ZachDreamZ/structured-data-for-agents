The Problem

AI agents need structured data from web pages. Every site has different HTML, anti-bot measures, and selector maintenance overhead. Building and maintaining scrapers for each site is expensive and fragile.

The Solution

Agent API Gateway is a REST API that extracts structured JSON from any public URL. Send a URL and a schema type (product, article, or company), and get validated JSON back — instantly.

# Extract product data from any URL
POST /v1/extract
{
  "url": "https://example.com/product-page",
  "schema_type": "product"
}

# Response: validated JSON with name, price, description, images, etc.

🔍 Schema-Guided Extraction

Three built-in schemas: product, article, and company. Each returns validated fields with proper types.

✅ Validated Output

Every response is validated against the schema. Missing fields are flagged. No guesswork.

🤖 Agent-Ready

Built-in llms.txt and agent.json for autonomous AI agent discovery.

⚡ Fast & Reliable

Deployed on Render with automatic scaling. 16,000+ requests served in the first 24 hours.

How It Works

1. Send a request — POST to /v1/extract with a URL and schema type.
2. Get structured JSON — Validated, typed, ready for your agent pipeline.
3. Handle errors gracefully — Clear error messages when extraction fails.

Supported Schemas

Product

Name, price, currency, description, images, availability, brand, SKU, ratings.

Article

Title, author, published date, content, word count, language, tags, image.

Company

Name, description, industry, employees, founded, headquarters, social links, contact.
